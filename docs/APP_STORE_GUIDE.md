## App Store Description (Hebrew)

**Subtitle:** מכשיר הקשר הפרטי שלך

**Promotional Text:**
שמוקי טוק - החוויה הדיגיטלית של מכשיר הקשר לחברים ומשפחה

**Description:**
שמוקי טוק מביא את הפשטות והמיידיות של מכשיר הקשר לעידן הדיגיטלי.

**תכונות עיקריות:**

🎤 **לחץ כדי לדבר**
לחץ והחזק את הכפתור המרכזי כדי לשדר - בדיוק כמו מכשיר קשר אמיתי.

👥 **חדרים פרטיים**
צור חדרים פרטיים לכל קבוצה - משפחה, חברים, קבוצת ספורט, נסיעה.

⚡ **תקשורת מיידית**
תקשורת קולית בזמן אמת עם השהיה מינימלית. לא צריך לחכות.

🔒 **פרטיות מלאה**
הקול שלך לעולם לא מוקלט. שיחה בזמן אמת בלבד.

📱 **מצב מאזין**
הצטרף כמאזין לחסוך בסוללה כשאין צורך לדבר.

🎯 **מערכת תור**
אם הערוץ תפוס - הצטרף לתור וקבל התראה כשתגיע תורך.

**כיצד זה עובד:**
1. היכנס עם חשבון Google שלך
2. צור חדר או הצטרף עם קוד הזמנה
3. שתף את הקוד עם חבריך ומשפחתך
4. לחץ כדי לדבר!

## Privacy Policy Required Points

- App collects: Google account name, email, profile photo
- App does NOT collect: voice content, location, contacts
- No voice recordings stored
- Firestore data encrypted at rest
- GDPR compliant: data deletion on account deletion
- Privacy Policy URL: https://shmukitalk.app/privacy

## App Review Notes

For Apple/Google review team:
- This app uses microphone for real-time voice communication only
- No voice is recorded or stored on any server
- WebRTC is used for peer-to-peer audio (similar to FaceTime/WhatsApp)
- Firebase handles user authentication and room management only
- Test account available upon request
